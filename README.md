# LLM_Semantic_Comparison
大语言模型中文语义理解能力横向对比测试报告
1. 项目简介
本项目为大语言模型课程作业，旨在通过阿里云 PAI-DSW 平台部署三款主流开源大语言模型（LLM），并针对中文语境下的歧义识别、复杂逻辑推理及高情境社交语用进行横向对比测试。
部署模型：
ChatGLM3-6B (智谱AI)
Qwen-7B-Chat (通义千问)
DeepSeek-LLM-7B-Chat (深度求索)
2. 环境配置
模型在非 GPU 环境（CPU-only）下稳定运行。
硬件平台：阿里云 DSW (规格：CPU 实例，32GB RAM)

5. 横向对比分析（核心报告）
5.1 语义解构能力
Qwen-7B-Chat 在处理“中国式语境”时展现了极高的预训练质量。对于多义词（如“意思”）在特定语境下的流变，其捕捉精度显著高于其他模型。
DeepSeek-7B 侧重于逻辑分析，其回答风格更接近于“语义指令集”，在处理语法成分拆解时效率极高。
5.2 逻辑建模能力
在多层嵌套逻辑（递归推理）测试中，三者均受限于 7B 参数规模，但 DeepSeek 展示了更好的稳定性，没有出现 ChatGLM3 的复读现象。这反映了 DeepSeek 在预训练数据中可能包含了更高质量的逻辑与数学语料。
3. CPU 环境下的部署表现
内存管理：32GB RAM 是 7B 模型 CPU 部署的“甜点”配置。使用 bfloat16 格式可有效平衡推理精度与内存安全。
工程兼容性：Qwen 和 DeepSeek 的代码结构更现代，对 transformers 库的适配性优于 ChatGLM3，后者在部署阶段需要较多的人工干预（补丁修复）。
6. 结论
综合各项测试，Qwen-7B-Chat 在中文语义理解深度和文化对齐方面表现最佳，是处理复杂中文交互任务的首选。DeepSeek-7B 则在逻辑一致性和部署工程化方面具有明显优势。
