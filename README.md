# LLM_Semantic_Comparison
# 大语言模型中文语义理解能力横向对比测试报告

> **项目说明**：本项目通过阿里云 PAI-DSW 平台，在 32GB CPU 环境下实现了三款主流 7B 级国产大语言模型的本地化部署，并完成了深度语义对比测试。

---

## 1. 项目简介
本项目旨在评估不同大语言模型（LLM）对中文复杂语境的理解深度。通过五个维度的极端语义测试，横向对比各模型在歧义识别、逻辑推理及社交语用方面的差异。

### 部署模型清单：
*   **ChatGLM3-6B**: 智谱AI出品，国产 LLM 的先驱。
*   **Qwen-7B-Chat**: 阿里巴巴通义千问，以强悍的中文理解著称。
*   **DeepSeek-LLM-7B-Chat**: 深度求索出品，逻辑推理与工程优化标杆。

## 4. 问答测试结果汇总
通过 5 个中文语义挑战题，对比三款模型的原生表现。

| 测试维度 | 测试题目 | Qwen-7B 表现 | ChatGLM3-6B 表现 | DeepSeek-7B 表现 |
| :--- | :--- | :--- | :--- | :--- |
| **词义歧义** | 冬天/夏天穿多少 | **极优**：识别 Max/Min 反转。 | **良**：解释混杂无关英文。 | **优**：逻辑拆解准确。 |
| **语态拆解** | 单身狗原因 | **极优**：分清主客观施受事。 | **中**：误判为标点符号问题。 | **优**：语义直击核心。 |
| **逻辑递归** | 他知道我知道... | **良**：尝试逻辑链条建模。 | **差**：陷入逻辑死循环。 | **优**：结构化推理最清晰。 |
| **语法关联** | 明明喜欢白白 | **优**：完美锁定主语。 | **良**：存在轻微幻觉。 | **优**：语法树分析准确。 |
| **社交语用** | “意思”的潜台词 | **极优**：完美对齐中式语境。 | **良**：仅提供词典解释。 | **良**：逻辑对但缺乏灵动。 |
